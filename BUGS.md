## Bug Report Template

When reporting bugs, use this format:

```markdown

**Status**: Open/Fixed
**Priority**: High/Medium/Low
**Commit**: [commit hash when fixed]

**Steps to reproduce**:
1. Step one
2. Step two
3. Step three

**Expected behavior**:
What should happen

**Actual behavior**:
What actually happens

**Root cause**:
What caused the bug (when known)

**Fix**:
How it was fixed (when fixed)
```
